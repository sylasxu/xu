# Pin Icons

This folder contains map pin icons for the xu miniprogram.

## Required Icons

1. `activity.png` - Default activity pin (orange, 30x36px)
2. `activity_gold.png` - Pin+ activity pin (gold, 45x54px, 1.5x size)
3. `activity_fire.png` - Boosted activity pin (with fire effect, 36x43px)
4. `ghost.png` - Ghost anchor pin (gray/translucent, 28x34px)
5. `cluster.png` - Cluster marker (circular, 40x40px)

## Design Guidelines

- Activity pins should be teardrop-shaped with a circular top
- Pin+ pins should be gold/yellow with a subtle glow effect
- Boosted pins should have a fire/flame element
- Ghost pins should appear semi-transparent
- Cluster markers should be circular with a number display area

## Color Palette

- Activity (default): #FF6B35 (Orange)
- Pin+ (gold): #FFD700 (Gold)
- Boosted: #FF4500 (OrangeRed) with flame
- Ghost: #9CA3AF (Gray) with 60% opacity
- Cluster: #FF6B35 (Orange) background
